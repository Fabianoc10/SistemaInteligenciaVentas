1. Propósito del proyecto
El proyecto Sistema de Inteligencia de Ventas con SQL, Python y Dashboard tiene como propósito transformar datos de ventas en información útil y accionable para apoyar la toma de decisiones comerciales. Se busca que el sistema permita identificar patrones, tendencias y métricas clave que respalden la gestión de inventario, la estrategia de ventas y la planificación comercial.

2. Objetivos específicos
Identificar productos de mayor desempeño en términos de volumen y rentabilidad.

Analizar el comportamiento de ventas en diferentes periodos (mensual, trimestral, anual) para detectar estacionalidad y variaciones significativas.

Generar reportes ejecutivos y dashboards interactivos que faciliten la interpretación de resultados y la toma de decisiones estratégicas.

Proveer recomendaciones basadas en datos para optimizar inventario y mejorar la estrategia comercial.

3. Preguntas clave de análisis
¿Qué productos tienen mejor desempeño en ventas?

¿Cómo varían las ventas según el periodo analizado (mes, trimestre, año)?

¿Qué tendencias se pueden identificar para planear el inventario de manera más eficiente?

¿Existen patrones de estacionalidad que impacten las ventas?

¿Qué segmentos de clientes o categorías de productos generan mayor valor?

4. Alcance del proyecto
Datos de entrada: registros de ventas, productos, clientes y periodos de tiempo.

Procesamiento: limpieza, transformación y análisis con SQL y Python.

Resultados: visualizaciones interactivas, reportes ejecutivos y dashboard accesible.

Limitaciones: análisis orientado a ventas históricas, sin modelos predictivos avanzados (nivel junior sobresaliente).
